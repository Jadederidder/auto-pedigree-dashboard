# Auto Pedigree Collections Dashboard

Monthly commission collections dashboard for Auto Pedigree.

## Live URL

> GitHub Pages requires either a public repo or a GitHub Pro/Team plan.
> To enable, either make this repo public or upgrade your GitHub plan,
> then go to **Settings → Pages → Source: `docs/` folder, branch: `main`**.
>
> Once enabled, the URL will be:
> `https://jadederidder.github.io/auto-pedigree-dashboard/`

## Details

| Field | Value |
|---|---|
| **Data source** | Revio Statement (SN Personal Code filter) |
| **Commission rates** | Single R24.50 · Family R38.50 |
| **Period** | July 2025 – March 2026 |
| **Built with** | Claude AI, Chart.js, SheetJS |
